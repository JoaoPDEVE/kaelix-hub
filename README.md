# Kaelix Hub - Premium Roblox Scripts Platform

> A premium futuristic website for selling advanced Roblox scripts, built with React, TypeScript, and modern web technologies.

## üé® About

Kaelix Hub is a premium platform for selling advanced Roblox scripts. The website features a futuristic dark design with purple neon accents, horizontal scroll navigation, and smooth animations that create an exclusive and professional experience.

## üöÄ Features

- **Horizontal Scroll Navigation** - Smooth lateral navigation between full-screen sections
- **Futuristic Design** - Dark theme with purple neon (#7d2cff) accents and glassmorphism effects
- **Interactive Cards** - Product and team cards with hover effects (glow, elevation, 3D transforms)
- **Animated Background** - Canvas-based particle animation system
- **Responsive Design** - Fully responsive with mobile hamburger menu
- **Bilingual Support** - Portuguese (BR) and English with language toggle
- **Modern Stack** - React 19, TypeScript, Tailwind CSS 4, Framer Motion, shadcn/ui
